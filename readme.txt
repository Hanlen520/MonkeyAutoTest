============================================ 使用说明 =========================================
使用方法:
    1.黑名单或者白名单仅在整机测试中有效.
    2.黑白名单不能同时使用(即whitlist.txt和blacklist.txt两个文件不能同时有内容)
    3.在黑白名单配置包名时每个包名占用一行,不能有对多余的空格及回车
    4.定制monkey测试配置包名每个包名一行
